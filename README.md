# CxBind Examples

pybind11 binding generator


## Clone
```bash
git clone https://github.com/crungelab/cxbind-examples
cd cxbind-examples
procure
```

## Virtual Environment
```bash
hatch shell
```

# Development

## Tool Chain

[cxbind](https://github.com/crungelab/cxbind)

[cxbuild](https://github.com/crungelab/cxbuild)

## Build

### Generate Bindings
```bash
cd examples/simple
cxbind
```

### Build
```bash
cxbuild
```
